# 📊 Sales Analysis Dashboard – Power BI

## 📌 Project Overview
The Sales Analysis Dashboard is an interactive Power BI project designed to analyze and visualize sales performance. It helps businesses track key metrics such as total sales, profit, quantity sold, and regional performance to support data-driven decision making.

This dashboard provides clear insights into sales trends, top-performing products, and customer segments.

## 🎯 Objectives
- Analyze overall sales performance
- Identify top-performing products and regions
- Track profit and revenue trends
- Provide insights for better business decisions

## 📊 Dashboard Features
- Total Sales, Profit, and Quantity KPIs
- Sales by Region visualization
- Top Products analysis
- Monthly Sales Trend
- Category-wise sales breakdown
- Interactive filters and slicers

## 🛠️ Tools & Technologies
- Power BI
- Microsoft Excel (Data Source)
- Data Cleaning & Data Visualization

## 📂 Dataset
The dataset includes information such as:
- Order ID
- Product Category
- Sales Amount
- Profit
- Region
- Order Date

## ▶️ How to Use
1. Download the `.pbix` file.
2. Open it using **Microsoft Power BI Desktop**.
3. Interact with filters and visualizations to explore sales insights.
